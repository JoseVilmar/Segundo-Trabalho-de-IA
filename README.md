# DBSCAN - Inteligência Artificial

Implementação do algoritmo **DBSCAN** desenvolvida do zero para a disciplina de Inteligência Artificial.  
O projeto realiza testes nas bases **Two Moons**, **Two Circles** e **Iris**, com visualização dos pontos classificados como **Núcleo**, **Borda** e **Ruído**.

---

**Criação do ambiente virtual (venv)**

No diretório raiz do projeto, execute:

```bash
python -m venv venv
```
**Ativação do ambiente virtual**

```bash
venv\Scripts\activate
```
**Instalação das dependências**
```bash
pip install numpy scipy matplotlib scikit-learn
```
**Execução do projeto**
```bash
python main.py
```
